# GelatoGo

GelatoGo je full-stack web aplikacija za pregled poslastičarnica i online naručivanje sladoleda, razvijena u okviru seminarskog rada.

## Tehnologije

* **Bekend:** Java, Spring Boot, Spring Security (JWT), Spring Data JPA, MySQL, Maven
* **Frontend:** React, React Router, Axios, HTML, CSS

## Funkcionalnosti aplikacije

* **Korisnički nivo:** Registracija i email verifikacija, JWT autentifikacija, upravljanje korpom, kreiranje narudžbina i uvid u istoriju kupovine.
* **Administratorski nivo:** Upravljanje narudžbinama (pregled i promena statusa) i CRUD operacije nad objektima (poslastičarnicama) i artiklima (sladoledima).
